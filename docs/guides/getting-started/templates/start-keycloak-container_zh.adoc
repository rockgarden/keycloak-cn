== 启动 {project_name}

在终端中输入以下命令启动 {project_name}：

[source,bash,subs="attributes+"]
----
{containerCommand} run -p 8080:8080 -e KEYCLOAK_ADMIN=admin -e KEYCLOAK_ADMIN_PASSWORD=admin quay.io/keycloak/keycloak:{version} start-dev
----

此命令将启动 {project_name} 在本地 8080 端口上的暴露，并创建一个初始管理员用户，用户名为 `admin`，密码为 `admin`。


[source,bash,subs="attributes+"]
----
{containerCommand} run -itd -p 8090:8080 -e KEYCLOAK_ADMIN=keycloak -e KEYCLOAK_ADMIN_PASSWORD=keycloak quay.io/keycloak/keycloak:{version} start-dev
----

此命令将以后台模式启动 {project_name}，在本地 8090 端口上的暴露，并创建一个初始管理员用户，用户名为 `keycloak`，密码为 `keycloak`。
